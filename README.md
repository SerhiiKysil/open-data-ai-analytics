1 Лабораторна робота з DevOps

Основною метою полягає аналіз тенденцій автомобільного ринку України на основі відкритих даних про реєстрацію транспортних засобів.

Джерело - Єдиний державний реєстр транспортних засобів(https://data.gov.ua/dataset/06779371-308f-42d7-895e-5a39833375f0), дані було взято саме за 2026 рік.

Питання для аналізу
1) Частка електромобілів (FUEL: ЕЛЕКТРО) становить більше 5% від усіх реєстрацій.
2) Найпопулярніший колір серед зареєстрованих автомобілів - білий.
3) Більшість зареєстрованих автомобілів випущені до 2015 року.
